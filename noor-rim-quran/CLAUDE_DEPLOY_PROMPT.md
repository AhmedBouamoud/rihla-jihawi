# مهمة Claude Code: نشر تطبيق نور ريم

المطلوب: نشر تطبيق `noor-rim-quran` كتطبيق ثابت Static App.

## خطوات ضرورية
1. افتح المجلد كاملاً.
2. لا تغيّر منطق التطبيق ولا النصوص القرآنية إلا بعد مراجعة بشرية.
3. انسخ صور ريم الحقيقية إلى:
   - `assets/rim/rim-hero.jpg`
   - `assets/rim/rim-reward.jpg`
   - اختيارياً: `rim-smile.jpg`, `rim-family.jpg`, `rim-football.jpg`
4. انسخ ملفات التلاوة MP3 إلى `assets/audio/` بالأسماء الموجودة في `README_AR.md`.
5. انشر على Netlify أو Vercel كموقع ثابت.
6. إن كان النشر على Netlify:
   - Build command: اتركها فارغة.
   - Publish directory: جذر المجلد الذي يحتوي `index.html`.
7. بعد النشر جرّب على الهاتف:
   - اختيار سورة.
   - زر سماع الآية.
   - زر تكرار.
   - تسجيل صوت ريم.
   - حفظ النجوم بعد إغلاق الصفحة وفتحها.

## ممنوعات مهمة
- لا تضف إعلانات.
- لا تضف روابط خارجية مشتتة.
- لا تستعمل TTS آلي لتلاوة القرآن.
- لا تجعل صور ريم خلف نص الآيات.
- لا تستعمل كلمة "خطأ" في التصحيح.

## تحسينات لاحقة مقترحة
- إضافة صوت الأب أحمد في عبارات التشجيع.
- إضافة شاشة شهادة PDF بسيطة بعد ختم كل سورة.
- إضافة وضع مراجعة أسبوعي.
- إضافة المزيد من السور القصيرة بعد نجاح النسخة الأولى مع ريم.
